===========================================
NOTES:
===========================================
- JSON does NOT support comments (//)
- Always use double quotes (") not single quotes (')
- Remember trailing commas are NOT allowed after last item
- All strings must be in quotes
- Numbers and booleans (true/false) don't need quotes
- Arrays use square brackets []
- Objects use curly braces {}

===========================================
QUICK TIPS:
===========================================
1. Validate your JSON at: https://jsonlint.com/
2. Keep JSON files properly formatted for readability
3. Use consistent naming conventions
4. Add all required fields for each data type
5. Empty arrays are valid: []
6. When adding new items, copy existing format
